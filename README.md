## Setup
Set up backend project (link below): 

```
https://github.com/AST-tnbt/ECG-Classifier-BE
```

Clone the project
```bash
git clone https://github.com/AST-tnbt/ecg-classification-web.git
```
Install dependences
```bash
npm i
```
Run the project
```bash
npm run dev
```
